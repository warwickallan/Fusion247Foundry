# Fusion247Foundry Agent Contract

This repository is Warwick's public Fusion247 Foundry. It is an ideas and work-package staging area, not the production Fusion247 system.

## Authority

- The repository remains public by Warwick's explicit decision.
- Do not create another Foundry repository.
- Do not merge migration branches unless Warwick explicitly approves the merge.
- Do not create GitHub issues for migrated ideas until the idea records have been reconciled against the ClickUp Current State pages.

## Scope Rules

- Treat `ideas/` as exploratory material.
- Do not browse unrelated idea folders, branches, issues, pull requests, or repository history to infer build scope.
- Build agents should act only from an approved build authority pack containing `BUILD-BRIEF.md`, `BUILD-CONTRACT.md`, and `IMPLEMENTATION-PLAN.md`.
- Escalate ambiguity instead of mining unrelated Foundry context.

## Migration Rules

- Preserve source provenance for migrated ClickUp material.
- Do not delete source information without an explicit disposition.
- Preserve Fable-authored informal pages as authored reviews or stress tests with attribution.
- Keep raw archive candidates until their content has been read and classified.

## Delivery Model

- BUILD-BRIEF: GPT plus Warwick.
- BUILD-CONTRACT / PRD / Master Prompt: GPT plus Warwick.
- IMPLEMENTATION-PLAN: Larry.
- There is no active HANDOFF.md document and no separate PRD.md document.
- Implementation work starts only from the approved three-document pack.
