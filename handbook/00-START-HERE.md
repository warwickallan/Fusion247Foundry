# Start Here

Fusion247 Foundry is the public idea, convergence, and production-handoff layer for Fusion247 work.

The repository exists to move work from exploratory idea capture into an approved build authority pack without contaminating production governance.

## Approved Delivery Pack

Every build moves through exactly three documents:

1. `BUILD-BRIEF.md` - GPT + Warwick produce the human-readable, executive-summary-first brief.
2. `BUILD-CONTRACT.md` - GPT + Warwick produce the complete PRD / Master Prompt and build authority.
3. `IMPLEMENTATION-PLAN.md` - Larry produces the concrete implementation method after examining the real target system.

There is no active `HANDOFF.md` document and no separate `PRD.md` document.

## Repository Map

- `handbook/` - operating rules and migration standards.
- `ideas/` - exploratory idea records and destination folders.
- `templates/` - templates for the three-document delivery pack and idea capture.
- `registers/` - current registers for ideas, decisions, and migration state.
- `archive/` - historical source material and migration records.
